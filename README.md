# SyncCode 🚀

A real-time collaborative code editor built using MERN Stack, Socket.IO, and CodeMirror.

Users can join rooms, collaborate live, write code together, and work in multiple programming languages in real time.

---

## ✨ Features

- 🔥 Real-time code collaboration
- 👥 Multi-user room support
- ⚡ WebSocket-based communication using Socket.IO
- 🧠 Auto-sync code on new user join
- 🌍 Multi-language support
- 🎨 Syntax highlighting with CodeMirror
- 👤 Live connected users list
- 📋 Copy Room ID feature
- 🚪 Leave room functionality
- 💻 Modern dark IDE UI
- 🔄 Realtime editor updates

## 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/synccode.git
```

---

## 2️⃣ Navigate to Project

```bash
cd synccode
```

---

## 3️⃣ Install Frontend Dependencies

```bash
yarn install
```

---

## 4️⃣ Install Backend Dependencies

Navigate to backend/server folder:

```bash
cd server
yarn install
```

---

## 5️⃣ Setup Environment Variables

Create `.env` file inside frontend:

```env
REACT_APP_BACKEND_URL=http://localhost:5000
```

---

## ▶️ Run Frontend

```bash
yarn start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

## ▶️ Run Backend

```bash
node server.js
```

Backend runs on:

```bash
http://localhost:5000
```
