# SyncCode 🚀

A real-time collaborative code editor built using MERN Stack, Socket.IO, and CodeMirror.

Users can join rooms, collaborate live, write code together, and work in multiple programming languages in real time.

---

## ✨ Features

- 🔥 Real-time code collaboration
- 👥 Multi-user room support
- ⚡ WebSocket-based communication using Socket.IO
- 🧠 Auto-sync code on new user join
- 🌍 Multi-language support
- 🎨 Syntax highlighting with CodeMirror
- 👤 Live connected users list
- 📋 Copy Room ID feature
- 🚪 Leave room functionality
- 💻 Modern dark IDE UI
- 🔄 Realtime editor updates

---
